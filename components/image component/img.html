Images
Images play an important role in any webpage... CSS plays a good role to control image display. You can set the following image properties using CSS
Responsive Image
To make an image responsive, you need to give a new value to its width property. Then the height of the image will adjust itself automatically
Round Image
The main CSS property responsible for the circular shape is the border-radius property. Setting the radius of the corners to 50% of the width/height results in a circle.
