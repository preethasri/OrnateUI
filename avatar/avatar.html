Avatar
Avatar is an profile image often used to provide additional information about the user
Normal Avatar
Normal avatar has rounded borders,fixed width and height it can be accessed using id:normal-img1
Square Avatar:
Avatar with square border is known as square avatar.it can be accessed using id:square-img1
Avatar With Different Sizes:
Avatar with different sizes.just setting the width and height differently.it can be accessed using id:avatar-small,avatar-large
Text Avatar
Avatar with text are offen used to indicate the person's name.it can be accessed using id:text-img1,text-img2
